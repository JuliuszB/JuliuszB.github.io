Car Sales Website

A semester long team project involving full-stack development of a car dealership website using the Play framework, allowing communication between the front and back end. Features included a log in system for users and administrators, browsing through items in the database and administrative features like add / update / delete items and users in the database from the website. As well as this, a functioning check out system and shopping cart was implemented. This project was undertaken using Agile Development methodologies with weekly Scrum meetings and Extreme Programming (XP)
